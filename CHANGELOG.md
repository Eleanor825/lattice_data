# Changelog

## 0.1.0

- Initialized the Lattice Phase 1 repository.
- Added a minimal schema family for domain data compilation.
- Added ingestion support for text, HTML, JSON, JSONL, and optional PDF.
- Added normalization, deduplication, and low-value filtering.
- Added dataset views for pretraining, QA, instruction, and knowledge records.
- Added manifest, dataset card, and source coverage reports.
- Added an end-to-end test and GitHub Actions CI.
